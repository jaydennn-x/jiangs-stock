# Task 000: 샘플 작업 파일

## 상태: 대기

## 설명

이 파일은 새 작업 파일 작성 시 참고할 수 있는 샘플 템플릿입니다.
실제 작업이 아니며, 새 작업 생성 시 이 구조를 따라 작성합니다.

## 관련 파일

- `src/app/example/page.tsx`
- `src/components/example-component.tsx`
- `src/lib/example-util.ts`

## 수락 기준

- [ ] 기능이 정상적으로 동작한다
- [ ] 반응형 디자인이 적용되었다
- [ ] TypeScript 타입 에러가 없다
- [ ] ESLint/Prettier 검사를 통과한다

## 구현 단계

### 1단계: 기본 구조 설정

- [ ] 필요한 파일 생성
- [ ] 타입 정의 추가

### 2단계: UI 구현

- [ ] 컴포넌트 마크업 작성
- [ ] 스타일링 적용
- [ ] 반응형 처리

### 3단계: 로직 구현

- [ ] 비즈니스 로직 구현
- [ ] API 연동 (해당 시)
- [ ] 에러 핸들링

> API 연동 및 비즈니스 로직 구현 작업에서 필수로 포함합니다.
